Installation Instructions

Option 1: Build it Yourself

sudo apt update
sudo apt install curl build-essential git tor
sudo systemctl enable --now tor.service
curl https://sh.rustup.rs -sSf | sh -s -- -y
git clone https://github.com/umutcamliyurt/Amnezichat.git
cd Amnezichat/client/
cargo build --release
cargo run --release

Option 2: Get it from GitHub Releases

If you prefer not to build it yourself, you can download precompiled binaries directly from the GitHub Releases page
